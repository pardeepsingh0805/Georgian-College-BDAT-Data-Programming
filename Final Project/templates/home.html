Welcome to Stock Stalker!

This website gets data for 5 most popular stocks including: "AAPL","MSFT","GOOG","TSLA" & "AMZN" and represent the information in graphical manner. This project is part of Learning at Georgian College, Barrie and this project is the final project for Data Programming. This website is not for official use. It fetch data every 24 hour for latest day and add it to the graph. Github repository is also available for this project.

API Info

Stock Stalker provides a variety of API for getting information from database. The information can be accessed for various purposes. To access the api please use the following information:

To get all information: ./api
To get information for a symbol: ./api/symbol
For a particular date./api/YYYY-MM-DD
For a particular date & symbol./api/YYYY-MM-DD/symbol

Note: Symbol represents the stock, so inorder to access api, please replace "symbol" with stock symbol. For eg: "./api/aapl" and for date "YYYY-MM-DD" pattern should be followed.
